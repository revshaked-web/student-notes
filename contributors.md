## 🎯 על הדף הזה

הפרויקט **student-notes** נבנה ומתוחזק בזכות קהילה של סטודנטים שתורמים מזמנם וידע. הדף הזה נועד להכיר את האנשים שעומדים מאחורי הפרויקט, את תפקידם ואת תחומי האחריות שלהם — ולעודד עוד סטודנטים להצטרף ולתרום.

## 🤝 כיצד להצטרף?

נשמח לתרומה נוספת מכל מי שמעוניין! כדי להצטרף:

1. **Fork** את הריפוזיטורי לחשבון ה-GitHub שלכם.
2. צרו **Branch** חדש עבור השינוי שלכם:
   ```bash
   git checkout -b feature/my-contribution
   ```
3. בצעו את השינויים הרצויים וודאו שהקוד עובד כראוי.
4. שלחו **Commit** עם הודעה ברורה:
   ```bash
   git commit -m "Add: short description of the change"
   ```
5. פתחו **Pull Request** מול ה-branch הראשי (`main`) עם הסבר קצר על מה שביצעתם.
6. אחד מהתורמים יבדוק את הבקשה ויחזור אליכם בהקדם 🙌

> 💡 **טיפ:** לפני שמתחילים, כדאי לבדוק את ה-Issues הפתוחים בפרויקט — ייתכן שכבר יש שם משימה מתאימה בשבילכם.
